Controls
1-6 - The order of magnitude (0-5, respectively) of particles to create simultaneously.
Q, W, E - Choose the representation.
Shift - Change to a random bright color.
R - Reset the system.
A, Mouse Left Click - Create particles for one iteration.
Space - Create particles continuously (every iteration).
Tab - Swap between the slowing rule sets to apply to the particles.
T - Iterate between all rule sets to apply to the particles.
Z - Enter debugging mode (used to get black and white screenshots of the particle explosion).

Introduction
This is the result of a research project for a Graphics Fundamentals course. The purpose was to create and maintain a particle explosion. Fulfilling that goal, the project explored how many particles could be maintained simultaneously and the different representations for particles. Some tangential explorations included how to reliably get a random pleasant color, how to map 2-D mouse clicks to a 3-D space, and how to calculate performance (i.e. frame rate).
